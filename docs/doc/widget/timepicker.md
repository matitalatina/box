# Timepicker

Popup widget for time object, based on [Flatpickr](https://flatpickr.js.org/)


#### Supported types
- **Time**

#### Params

| Key          | Value             | Default           |
|:-------------|:------------------|:------------------|
| fullWidth    | show the picker at 100% width | `false`  |
